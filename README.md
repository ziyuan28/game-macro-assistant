# 🛠️ 自动化助手  - 硬件级游戏按键助手


这是一款专门针对 3D 动作游戏的高级按键宏工具。它采用了内核级硬件扫描码技术，能够有效绕过常规软件层面的脚本检测。

---

## ✨ 核心特性

- **🛡️ 硬件级模拟**：基于 Windows `ctypes` 底层 API，发送真实的硬件扫描码（Scan Code），让系统认为指令来自真实键盘。
- **🎮 3D 游戏兼容**：针对 3D 引擎优化，支持 DirectInput 模式，解决“记事本有用，游戏里没反应”的痛点。
- **🎨 现代电竞 UI**：采用 `CustomTkinter` 暗黑主题设计，界面硬核且直观。
- **📝 纯中文脚本解析**：首创小白友好型语法，直接用“按键”、“等待”等中文字符编写逻辑。
- **📁 多脚本管理**：支持本地 JSON 存储，一键保存、切换、删除多个连招配置。
- **⚡ 权限自动接管**：内置 UAC 管理员权限请求，确保对高权限游戏窗口有效。

---

## 🚀 语法支持

工具内置了傻瓜式语法引擎，支持以下指令：

| 指令 | 示例 | 说明 |
| :--- | :--- | :--- |
| **按键** | `按键 4` | 模拟按下并抬起 4 键，自带随机按压延迟 |
| **等待** | `等待 24` | 延迟执行，自带 ±5% 的时间扰动（防检测） |
| **左键** | `左键` | 模拟鼠标左键点击 |
| **右键** | `右键` | 模拟鼠标右键点击 |
| **长按** | `长按 W` | 持续按下某个按键 |
| **松开** | `松开 W` | 释放正在长按的按键 |
